bash-4.3$ python genos2sat -h
usage: genos2sat [-h] [-r R] [-i] [-s] [-n N] loc

Genotype data to SAT formula converter.

positional arguments:
  loc         file or directory containing genotype data

optional arguments:
  -h, --help  show this help message and exit
  -r R        possible number of haplotypes
  -i          include incompatibility extension
  -s          include symmetry breaking extension
  -n N        maximum number of genotypes to include
